Rujta – Smart Pharmacy Management & Medicine Ordering System
Overview
Rujta is a complete pharmacy management and medicine ordering system designed to digitally transform how pharmacies operate and how users access medications.

The system connects customers with nearby pharmacies through a modern mobile application while providing pharmacies with a powerful management dashboard. Users can search for medicines, upload prescriptions, place orders, and complete secure online payments. Pharmacies can efficiently manage inventory, update pricing, and track incoming orders in real time.

Rujta aims to create a seamless, reliable, and scalable digital healthcare experience.


User Features

Search medicines by name.
Add medicines to cart.
Place orders directly from the app.
Secure online payment with transaction validation.


Pharmacy Features

Add, update, and delete medicines.
Manage stock levels and inventory.
Update medicine prices.
View and manage customer orders.
Track completed and pending payments.

Tech Stack

Frontend: Flutter
Backend: Node.js
Database: MySQL
